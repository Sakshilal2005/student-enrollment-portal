# Student Enrollment Portal

A full-stack web application for managing college admissions.

##  Features

* Student registration form
* Stores data in database
* Displays available courses
* Simple and clean UI

##  Tech Stack

* Frontend: React
* Backend: Flask (Python)
* Database: SQLite

##  How to Run Project

### Frontend

```bash
npm install
npm start
```

### Backend

```bash
pip install -r requirements.txt
python app.py
```

##  Project Description

This project allows students to apply for courses through an online form. The data is stored in a database and can be used for admission processing.
